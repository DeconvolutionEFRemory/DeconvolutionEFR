# DeconvolutionEFR
Implementation of the Leray-alpha-EFR model for incompressible flows. The work is supported by the National Science Foundation through grants NSF-DMS1620406/162038 (PIs: Alessandro Veneziani and Annalisa Quaini) and TG-ASC160069 (PI: Alessandro Veneziani).
